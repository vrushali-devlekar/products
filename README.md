# 🛒 JavaScript Callback Practice – Product Cart App

A simple **frontend project** built to practice **JavaScript callbacks, Fetch API, DOM manipulation, and localStorage** by creating a mini **product listing and cart system**.

The application fetches products from an API, displays them in a grid layout, and allows users to add products to a cart and view them on a separate cart page.

---

## 🚀 Features

- Fetch products from a public API
- Dynamic product rendering using JavaScript
- Add product to cart
- Store cart data in **localStorage**
- Redirect to a **cart page**
- Display selected product details
- Responsive product grid layout
- Practice **callback functions**

---

## ⚙️ How It Works

### 1️⃣ Load Products

Products are fetched from a public API:

JavaScript dynamically creates product cards and displays them on the page.

---

### 2️⃣ Add To Cart

When the **Add To Cart** button is clicked:

1. Product is added to the cart array
2. Total price is calculated
3. Cart is saved to **localStorage**
4. User is redirected to **cart.html**

---

### 3️⃣ Cart Page

The cart page:

- Reads cart data from **localStorage**
- Displays product title
- Displays product image
- Displays product price
